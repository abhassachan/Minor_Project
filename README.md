# Minor Project — GPS-Enabled Gamified Running Web Application

## Project Overview
This project aims to develop a GPS-enabled gamified running Progressive Web Application (PWA) that transforms outdoor running into an engaging and competitive experience. The system tracks user movement in real time and allows users to claim virtual territories based on their running routes.

## Key Features
- Real-time GPS tracking  
- Interactive map-based route visualization  
- Territory claiming mechanism  
- Basic user competition  
- Mobile-first Progressive Web App  

## Planned Tech Stack
Frontend: React.js  
Backend: Node.js + Express  
Database: MongoDB  
Maps & GPS: Leaflet / Browser Geolocation API
